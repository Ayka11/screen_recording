# Submission Checklist (Lignum)

## 1) Account and Console Setup
- [ ] Google Play Console account created and verified
- [ ] App created in Play Console (`com.ayka.lignum`)
- [ ] Contact email/website/phone configured

## 2) Policy and Legal
- [ ] Privacy policy URL available and added in Play Console
- [ ] App content declarations completed
- [ ] Data Safety form completed (see `data-safety-declarations.md`)
- [ ] Content rating questionnaire completed (see `content-rating-guide.md`)
- [ ] Target audience and ads declarations completed

## 3) App Quality / Technical
- [x] Signed AAB generated
- [x] Release build successful on Gradle
- [x] No external payment flow in build
- [ ] Real-device smoke tests completed (Android 10, 12, 14+)
- [ ] App icon, feature graphic, screenshots exported and uploaded

## 4) Testing Track
- [ ] Closed testing configured
- [ ] Testers added and opted in
- [ ] Feedback pass completed
- [ ] Production access requested after testing requirements are met

## 5) Production Rollout
- [ ] Upload AAB to production release
- [ ] Add release notes
- [ ] Resolve all Play pre-launch report blockers
- [ ] Start staged rollout (recommended 10-20%)
- [ ] Monitor crashes/ANRs after launch
