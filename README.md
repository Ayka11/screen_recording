# Lignum

A micro-learning game and encyclopedia about trees.

## Run locally

Prerequisites: Node.js

1. Install dependencies: `npm install`
2. Set `GEMINI_API_KEY` in `.env.local`
3. Start dev server: `npm run dev`

## Android (Capacitor)

1. Build web assets: `npm run build:web`
2. Sync to Android: `npm run cap:sync`
3. Build AAB: `npm run android:build:aab`

Release and Play submission docs are in `docs/play-store/`.
